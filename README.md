# Memory Lane App

## Description 

Memory Lane is a web application designed to help users with/at risk of Alzheimer's disease preserve and reinforce memory through a simple and accessible digital platform.
This project was developed as part of the 2025 Congressional App Challenge and focuses on user experience and real-world problem solving. Memory Lane is deployed as a live application using Vercel and demonstrates end-to-end development from concept to implementation.

The app offers: 
 - Custom photo gallery
 - To-do list
 - Daily notes
 - 3 brain stimulating memory games
 - Detailed information page with helpful videos and links
 - Calming music proven to help Alzheimer's patients
 - Accessibility features

Link to project: memorylaneapp.vercel.app
